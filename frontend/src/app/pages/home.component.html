Chess Agent

Welcome to the Chess Agent demo. This project is built as a portfolio-ready showcase of an AI-assisted decision workflow for chess.

Explore the different demo pages to play on an interactive board, analyze FEN positions with a chess engine, search openings with a RAG-style vector search, and retrieve curated YouTube videos related to an opening.

FEN (Forsyth-Edwards Notation) is a standard string that encodes a full chess position: pieces, side to move, castling, en passant, and move counters.

Technical note: the frontend is built with Angular and Bootstrap. The backend is a Python API (containerized with Docker) that integrates a Stockfish-based engine for analysis, a Milvus vector database for embeddings + retrieval, and the YouTube Data API for video search.
